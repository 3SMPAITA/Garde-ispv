# Garde ISPV — Guide de déploiement

## Ce que contient ce dossier
- `src/App.js` — L'application complète
- `src/firebase.js` — La connexion à votre base de données Firebase
- `src/index.js` — Point d'entrée
- `public/index.html` — Page HTML
- `package.json` — Dépendances
- `netlify.toml` — Configuration Netlify

## Pour déployer sur Netlify (via GitHub)

1. Créez un dépôt sur github.com (bouton "New repository", nom : garde-ispv)
2. Uploadez tous ces fichiers dans le dépôt
3. Sur netlify.com → "Add new site" → "Import from Git" → choisissez votre dépôt GitHub
4. Netlify détecte automatiquement la config, cliquez "Deploy"
5. Dans quelques minutes vous avez votre lien URL !
